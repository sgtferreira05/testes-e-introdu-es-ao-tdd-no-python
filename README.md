## ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/sgtferreira05/Testes-e-introdu-es-ao-TDD-no-Python.git
```

Entre na pasta:

```bash
cd Testes-e-introdu-es-ao-TDD-no-Python
```

Crie o ambiente virtual:

```bash
python -m venv venv
```

Ative o ambiente virtual:

### Windows
```bash
venv\Scripts\activate
```

### Linux/macOS
```bash
source venv/bin/activate
```

Instale as dependências:

```bash
pip install pytest
```

Execute os testes:

```bash
pytest
```
